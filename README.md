# Optimizacion-de-gastos-de-marketing

Este es un proyecto de optimizacíon de gastos de marketing de una empresa ficticia de venta de entradas a eventos.

En el proyecto se visualizaron el uso del servicio de parte de los clientes, cuando comenzaron a comprar y cuanto dinero aporta cada cliente.

En este proyecto se encuentra con los siguientes datos:

* Registros del servidor con datos de las visitas a showz desde enero 2017 a diciembre 2018

* Un archivo con los pedidos en dicho periodo

* Estadísticas de gastos de markerting
